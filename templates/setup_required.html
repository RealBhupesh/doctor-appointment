Database Setup Required

This app is deployed on Vercel but no PostgreSQL database is connected. SQLite cannot persist data across serverless requests, so appointments and updates will not work correctly.

How to fix

  1. Open your project in the Vercel Dashboard
  2. Go to Storage → Create Database → Postgres
  3. Connect the database to this project
  4. Redeploy (or wait for auto-deploy)

The POSTGRES_URL environment variable will be set automatically when you connect the database.
